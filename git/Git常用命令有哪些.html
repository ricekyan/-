Git常用命令有哪些

        git config --global user.name 提交人姓名
        git config --global user.email 提交人邮箱 
        Git config --list 查看git配置信息
        Git init 初始化git仓库  git status查看文件状态 
        Git add +文件名 添加到暂存区
        Git commit 提交文件 git commit -m 第几次提交
        Git log 查看历史提交记录
        Git checkout  用暂存文件覆盖工作目录中的文件
        Git rest --hard commitID 将git仓库中的文件恢复出来
        Git branch 查看分支 git branch develop 创建分支
        Git checkout +分支名  切换分支
        Git branch -d develop 删除开发分支 -D是强制删除Git push +远程仓库名 master 将本地仓库推送到远程仓库
        Git remote add +自定义名称 +远程仓库地址 （简化远程仓库名）
        git pull name branch 拉取名字为name的远程仓库的branch分支
        git clone 地址 克隆项目到本地
