Dom对象与JQuery对象有什么区别？

        
                ​  jquery对象是DOM对象通过jquery包装后的产物，是jquery独有的。

                ​  jquery里的方法只有jquery对象才可以使用。
                
                    二者之间可以相互转换：

                    DOM 对象转 jQuery 对象:
                    var oDiv = document.querySelector('div'); // DOM 对象
                    DOM 对象转 jQuery 对象，用 $() 包起来
                    $(oDiv).css('background', 'red');
               
                
                    jQuery 对象转 DOM 对象:
                    var $div = $("div"); // jQuery 对象
                    $div[0]
                    $div.get(0)
